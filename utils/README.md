### BinarySearchTree （二叉搜索树）
#### 1. 概念
二叉搜索树具有一下四点性质：
+ 所有节点关键码都互不相同
+ 左子树上所有节点的关键码都小于根节点的关键码
+ 右子树上所有节点的关键码都大于根节点的关键码
+ 左右子树也是二叉搜索树
